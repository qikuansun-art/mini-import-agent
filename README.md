# Mini Import Agent

An AI Agent for Excel Import, Knowledge Retrieval and Workflow Automation.

---

## Project Background

Traditional Excel import usually requires:

- Manual field mapping
- Rule checking
- Data validation
- User confirmation

This project uses:

- Agent
- Workflow
- RAG
- LLM

to automate the import process.

---

## Workflow

```text
User Upload Excel

↓

Excel Parser

↓

Field Analysis

↓

Knowledge Retrieval

↓

LLM Understanding

↓

Workflow Validation

↓

User Confirmation

↓

Import System

↓

Generate Report
```
